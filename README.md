# AWS/DevOps Learning System

## Strategy

### Phase 1: AWS Academy (NOW)
- Use AWS Academy as primary learning source
- Go module by module sequentially
- Complete assignments and ALL labs seriously
- Focus on understanding + structured coverage

### Phase 2: AWS Skill Builder (LATER)
- Activate after finishing core modules
- Use for: official practice exams, deeper learning, certification prep
- Unlock free certification vouchers

## Constraints
- 6-month time window
- Mom's AWS account (free tier for hands-on)
- Personal account (IAM Identity Center practice)
- Prioritize depth over breadth

## Structure

```
01-syllabus/          - AWS SAA-C03 exam checklist
02-daily/            - Daily templates, trackers, kanban
03-notes/            - Topic notes (iam/, s3/, vpc/)
04-practice/         - Hands-on labs, exercises
05-progress/         - Dashboards, reviews
11-aws-academy/      - AWS Academy course notes
```

## Daily Mission
1. Complete 1 AWS topic (60 min)
2. Build/Practice what you learned
3. Record explanation (communication practice)
